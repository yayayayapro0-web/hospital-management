# نظام إدارة المستشفى

## وصف المشروع
نظام ويب لإدارة المستشفى يتيح تسجيل وإدارة المرضى والأطباء والمواعيد والأقسام.

## فكرة المشروع
توفير منصة بسيطة لإدارة بيانات المستشفى مع لوحة تحكم للمدير.

## طريقة تشغيله
1. ثبّت XAMPP وشغّل Apache و MySQL
2. انسخ المشروع داخل `C:\xampp\htdocs\hospital`
3. افتح المتصفح واكتب `http://localhost/hospital/setup.php`
4. افتح `http://localhost/hospital/`
